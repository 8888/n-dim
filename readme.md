You are navigating a 4-dimensional space, but only have 2-dimensional slices of a plane through the space as your view.

![alt text](image.png)

## Controls

| Key | Action |
|---|---|
| a | Move -1 in the x dimension |
| q | Move +1 in the x dimension |
| s | Move -1 in the y dimension |
| w | Move +1 in the y dimension |
| d | Move -1 in the z dimension |
| e | Move +1 in the z dimension |
| f or z | Move -1 in the w dimension |
| r or c | Move +1 in the w dimension |
| click | Inspect a point |
